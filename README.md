# FastAPI Notes API

A simple CRUD REST API built with FastAPI, PostgreSQL, and SQLAlchemy.

## Features
- Create, read, update notes
- PostgreSQL integration
- Pydantic validation
- Clean architecture

## Tech Stack
- FastAPI
- SQLAlchemy
- PostgreSQL
- Pydantic

## Setup
- python -m venv venv
- venv\Scripts\activate
- pip install -r requirements.txt
- uvicorn app.main:app --reload

## Swagger UI available at:
- http://127.0.0.1:8000/docs
